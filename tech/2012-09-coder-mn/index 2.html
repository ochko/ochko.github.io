Кодер сайтын 3 дахь удаагийн шинэчлэл
2012-09-18

Кодер сайт маань 3 дахь удаагийн дорвитой өөрчлөлтүүдийг бэлэн болохыг хүлээгээд түр хугаацаанд ажиллагаагаа завсарлав. Сэрвэрээ нүүлгэхийн зэрэгцээ бодлого шүүлт, шалгах хэсгүүд дээр зарчмын өөрчлөлтүүд орохоор төлөвлөгдсөн. Энэ хугацаан кодерууд маань бэлтгэлээ хангаад бодлогоо бэлдээд байцгааж байгаарай. Хөгжүүлэлтэд оролцох, дараагийн өөрчлөлтөд өөрийн санал хүсэлтийг оруулахыг хүсвэл манай сайтын эх код github дээр нээлттэй байгаа гэдгийг санаарай.
