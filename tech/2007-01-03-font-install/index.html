Линукс дээр Виндовсын фонт суулгах

2007-01-03

Яагаад Linux дээр Windows фонтууд суулгах хэрэгтэй болдог вэ?

  • Зарим вэб хуудсуудыг Виндовсын фонтоор үзэхээр тохируулаад дизайныг хийчихсэн байдаг. Туршлагагүй вэбчидийн хийсэн хуудас фонд солигдоход хэлбэр галбираа алдаж эвгүй харагдах гэх мэт тохиолдол бий.
  • MS Word, MS Powerpoint зэрэг програм дээр бэлдсэн материалыг OpenOffice зэрэг бусад офисын програм дээр нээхэд хэлбэр загвар нь эвдэрч харагдах тохиолдол гардаг. Ийм файлуудыг эвтэйхэн анхны байдлаар үзэхийн тулд анх хийгдсэн фонтууд нь зайлшгүй хэрэгтэй болно.
  • Үнэндээ MS Windows фонтууд загвар сайтай, ялангуяа жижигхэнээр харахад хамгийн зөв хэлбэртэй харагддаг. Яагаад гэвэл майкрософтод зөвхөн фонтны дизайн хийгээд, сайжруулаад сууж байдаг дизайнерууд байдаг учраас тэр. Фонтыг үсэг үсгээр нь нарийн шинжиж, хэдэн цэгийн өндөртэй байхад яаж харагдуулахав, ямар үсэгний дараа хэдэн цэгийн зайтай гарч ирэхэв гэх мэтчихлэн маш сайн тооцож хийсэн байдаг.

Гэхдээ MS фонтуудыг Linux дээр аваад ашиглахад хууль зөрчих үү?

  • Анх MS Internet Explorer-ийн Netscape тэй өрсөлдөж байх үед өөрийнхөө бүтээгдэхүүнд бага ч гэсэн давуу тал авчрах зорилгоор майкософтчууд өргөн хэрэглэж байсан truetype фонтуудаа хэрэглэхэд үнэгүй лизэнзтэйгээр вэб сайт дээрээ татаж авах боломжтой тавьж байсан байна. Тэдгээр фонтууд нь:
  1. Andale Mono
  2. Arial Black
  3. Arial (Bold, Italic, Bold Italic)
  4. Comic Sans MS (Bold)
  5. Courier New (Bold, Italic, Bold Italic)
  6. Georgia (Bold, Italic, Bold Italic)
  7. Impact
  8. Times New Roman (Bold, Italic, Bold Italic)
  9. Trebuchet (Bold, Italic, Bold Italic)
  10. Verdana (Bold, Italic, Bold Italic)
  11. Webdings

Харин хэдэн жилийн дараа IE нь Netscape аас илт давуу зах зээл эзлээд ирэнгүүт вэб сайт дээрээ үнэгүй тавьсан байсан фонтуудаа татан буулгасан боловч лизенз нь тэр хэвээрээ үлдсэн учраас одоо болтол эдгээр фонтуудыг үнэгүй ашиглаж болдог юм байна.

Фонт суулгах хэдэн аргыг тайлбарлая.

  1. nautilus ашиглаж байгаа бол шууд фонтын файлуудыг fonts:// байрлалд хуулахад болно.
  2. хэрэв root эрх байхгүй бол өөрийн home директори дотроо .fonts гэж директори үүсгээд фонтуудыг хуулж болно.
  3. линуксынхаа багц модуль суулгах системийг ашиглаж болно.

-debian, ubuntu дээр бол: apt-get install msttcorefonts -slackware дээр бол: slapt-get msttcorefonts -бусад линуксууд дээр rpm эсвэл багц файлыг нь татаж аваад суулгаж бас болно.

Ингээд суулгахаар Виндовсын хамгийн гоё фонт болох Tahoma энэ багц дотор байхгүй байгаа, учир нь Tahoma бол бүр сүүлд хийгдсэн фонт учраас чөлөөтэй хэрэглэх тэрхүү лицензэд багтдаггүй юм. Иймд та бид дур мэдээд Tahoma гэх мэт дуртай truetype фонтоо дараах аргаар суулгаж бас болно.

  1. линуксынхаа truetype фонтуудын байгаа байрлалд суулгах фонтуудаа(tahoma.ttf, tahomabd.ttf) нэгэн шинэ директори(windowsfonts) үүсгээд хуулна. ихэвчлэн /usr/share/fonts/truetype/ дотор байдаг.
  2. тэр дирэктори дотроо ороод mkfontdir командыг ажиллуулна.
  3. /usr/share/fonts/truetype/fonts.cache-1 файл дотор "windowsfonts" 0 ".dir" гэсэн мөр нэмж өгнө("windowsfonts" гэдэг нь шинээр үүсгэсэн директорийн нэр).
  4. fc-cache командыг ажиллуулна.

Тэгээд систэмийнхээ фонтыг Tahoma болгоод, хэмжээг нь 8 дээр тавиад, антиалиасийг идэвхгүй болгочихоод харахаар цонхнууд чинь цэмцийгээд гоё харагдаж байгаа биз.
