Интэрфэйс дизайн
2007-01-04

Интэрфэйс бол хоёр тусдаа зүйлийг хооронд нь холбодог эд билээ. Хоёр тал холбогдохын тулд аль аль нь интэрфэйстэй байгаад тэдгээр нь хоорондоо ойлголцдог байх ёстой. Компьютерыг компьютертэй холбодог сүлжээний карт, програмыг програмтай холбодог RMI, CORBA, TCP интэрфэйсүүд, програмыг хүнтэй холбодог график хэрэглэгчийн интэрфэйс, хүнийг комьпютэртэй холбодог хэл яриа хөрвүүлэгч програм, эсвэл хулгана, гар, дэлгэц гэх мэт бүгдээрээ л интэрфэйс юм. Интэрфэйсийн дизайныг хийхэд баримтлах гол зарчим бол гаднаас харахад ойлгомжтой энгийн байлгах юм. Гаднаас интэрфэйсийг ашиглагч талд дотоод үйлдлүүд болон дотоод бүтэц хамаагүй боловч дотоод мөн чанаруудаа орхигдуулалгүй илэрхийлж чадахуйц байх нь бас чухал.

Хэрэглэгчийн график интэрфэйсийн хувьд аль болох энгийн байлгах хэрэгтэй гэжээ. Гэвч энгийн байлгах гээд цөөхөн функцтэй ядуу програм байж болохгүй. Энгийн мөртлөө олон функцтэй, хэрэглэхэд эвтэйхэн байлгахын тулд програмаа функцүүдийн цуглуулга биш, цогц үйлүүдийн урсгал байхаар загварчлах хэрэгтэй. Үнэхээр сайн дизайн хийж чадвал хэрэглэгч хүссэн зүйлээ бодож хайлгүйгээр шууд зөнгөөрөө л хийчихнэ гэсэн үг. Хэрэглэгчийг олон сонголтуудаас сонгуулж бодуулах нь сайн шийдэл биш гэнэ. Нэг удаагийн хулганы товшилт, хэрэглэгчийн нэг удаагийн нүдний хөдөлгөөн ч үнэтэй бөгөөд эдгээрийг тооцож чадвал хүн бүхэн вэбийг чинь хэрэглэх дуртай байх болно.

Хэрэглэгчийн интэрфэйстай холбоотэй вэб холбоосууд:

  • The Do’s and Don’ts of Web Site Design. Вэб дизайн хийхэд юуг хйих, юунаас зайлсхийх хэрэгтэйг заасан байна.

  • Simplicity. Энгийн байлгана гэдэг дизайныг сайжруулж байгаа боловч дутуу функцтэй байна гэдэг муу дизайнтай байхаас ч дор.

  • Web Design Features. Сайн дизайнтай вэб ямар шинж чанаруудтай байдаг, муу дизайнтай вэб ямар шинжтэй байдгийг тоочсон байна.

  • Web Site Usability Checklist. Хэрэглэхэд хялбар вэб ямар нөхцөлүүдийг хангаж байх ёстойг шалгах шинжүүрүүдийн жагсаалт.

  • Criteria for optimal web design. Зураг хэрхэн ашиглах, хөгшид хүүхдүүдэд хэрэглэхэд хялбар байлгах, интернэйшнл байлгах гэх зэрэг зөндөө зүйлс байна.

  • User Interface Design for Programmers. Яаж төсөөлсөн түүгээр нь ажиллах дизайныг үзүүлж чадвал хэрглэгчдийн сэтгэл ханах болно.
