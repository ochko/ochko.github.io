Програмчлалын номууд

2006-09-17

Ширээний ном

Надад ер нь бүтнээр нь уншсан ном гэж байдаггүй юм байна. Шинэ ном олохоороо зүгээр л гарчиглаад орхичихно. Дараа нь лавлах юм гарвал харин хэрэгтэй хэсгээ нухацтайхан уншчихдаг юм. Програмчлалын холбогдолтой номыг ингэж унших нь үр дүнтэй ч байж магадгүй. Програм бичих хугацааны нилээд хувь нь гарын авлага, help сөхөж харах зэрэгт зарцуулагддаг шүү дээ. Миний байнга сөхөж харж байдаг номнууд энэ байна. Заримыг нь гарчиглаад орхисон, заримынх нь чухал хэсгүүдийг уншчихсан гэхэд болно.

Програмчлал

  • Steve McConnell. Code Complete, Second Edition. Сонгодог зохиол тул заавал уншаарай.

  • Martin Fowler. Refactoring: Improving the Design of Existing Code. Энэ номыг уншсан сэтгэл хөнгөрөх шиг болсон шүү.

  • Andrew Hunt, David Thomas. The Pragmatic Programmer: From Journeyman to Master. Зарим хэсгээс уншсан. Сайхан зөвлөгөө олон байгаа.

  • Frederick P. Brooks. The Mythical Man-Month: Essays on Software Engineering, 20th Anniversary Edition. Ганц нэг шад уншсан. Сонгодог ном.

  • Erich Gamma. Design Patterns: Elements of Reusable Object-Oriented Software. Загварчлалын хэвийг ойлгохыг хүсвэл уншаарай.

Unix програмчлал

  • Eric S. Raymond. The Art of UNIX Programming. Юникс програмчлал гэж юу вэ гэвэл энэ. Ерөнхийд нь гүйлгээд харсан.
  • W. Richard Stevens. Advanced Programming in the UNIX Environment. Юникс програмчлалд хөл тавьж байгаа бол нэг гартаа бариад суухаас аргагүй ном.
  • Kay Robbins, Steve Robbins. Unix Systems Programming: Communication, Concurrency and Threads. Дээрхтэй давхцах юм бий. Гэхдээ хоёулаа нийлэхээрээ бүр янзтай.
  • Brian W. Kernighan, Rob Pike. The UNIX Programming Environment. Юникс програмчлалын орчинд хэрэглэгддэг хэрэгслүүд зэргийг тайлбарласан байгаа.
  • Peter Prinz, Tony Crawford. C in a Nutshell. C хэлний толь бичиг. Мөн үүний оронд The C Programming Language ийг хэрэглэсэн ч болно.

Java програмчлал

  • David Flanagan. Java In A Nutshell. Их ойлгомжтой бичигдсэн байгаа. Анх сурж байгаа хүнд зүгээр.

  • Joshua Bloch. Effective Java Programming Language Guide. Анхан шатны мэдлэгтэй болсны дараа уншсан нь дээр.

  • Brian Goetz. Java Concurrency in Practice. Энэ номонд орсон олон зүйлсийг дараа нь Жава 5 хувилбарт оруулсан.

  • Rod Johnson. Expert One-on-One J2EE Design and Development. J2EE ийн тухай ойлгомжтой бичигдсэн ном.

  • Steven John Metsker. Design Patterns in Java. Энэ номыг уншаагүй. Гэхдээ гарчигласан.

Та харин ямар ном санал болгомоор байна?
